# **App Name**: Studio Forty-Five Landing

## Core Features:

- Hero Section: Displays a striking title, elegant subtitle, and a prominent call-to-action button to solicit quotes.
- Concept Gallery: A clean grid layout showcasing photos of Studio 45's porcelain artistry, emphasizing a frameless, minimalist aesthetic.
- Differentiators Display: Presents key differentiators using a minimal list format with refined text and numbering to highlight precision and craftsmanship.
- WhatsApp Integration: Incorporates a discreet, always-available WhatsApp button for direct client communication, streamlining quote requests.
- Subtle Background Animation: Uses CSS or Canvas to generate a subtle, angled line animation resembling stone veins, reinforcing Studio 45's brand identity with a minimalist touch.

## Style Guidelines:

- Primary color: Graphite Gray (#333333) for text elements to convey sophistication.
- Background color: Absolute White (#FFFFFF) with alternating sections in Ice Gray (#F9F9F9) for a clean, premium feel.
- Accent color: Titanium Gray (#546E7A) for buttons, offering a subtle, technical contrast.
- Headline font: 'Montserrat' (sans-serif) at weight 700 for strong, clear titles. Note: currently only Google Fonts are supported.
- Body font: 'Inter' (sans-serif) at weight 300 for elegant, readable text. Note: currently only Google Fonts are supported.
- Use a layout emphasizing spaciousness, sharp corners, and the removal of visual clutter to achieve a minimalist aesthetic.
- Incorporate a subtle fade-in effect for sections upon scrolling and transition effects on buttons, using a transition time of 0.3 seconds with an ease timing function.
- Use a minimalist gray icon for a WhatsApp button that remains fixed for easy access.  Consider providing a transition as the user hovers over the element.
